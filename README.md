# Cognifyz Technologies Internship Tasks

This repository contains all tasks completed during the Cognifyz Technologies Frontend Development Internship.

## Tasks Included
- Task 1 – HTML Practice
- Task 2 – Forms & Layout
- Task 3 – CSS Styling
- Task 4 – JavaScript Basics
- Task 5 – DOM Manipulation
- Task 6 – Weather App
- Task 7 – Responsive Design
- Task 8 – SCSS & Advanced Styling

## How to Run
Open any `index.html` file in a browser to view the project.

## Task Descriptions

### **Task 1 – Basic HTML Page**
- Created a simple webpage with headings, paragraphs, and an image.
- Practiced HTML structure and basic elements.

### **Task 2 – Inline CSS Styling**
- Applied inline CSS directly inside HTML elements.
- Styled text, background, and image elements.

### **Task 3 – Responsive Design**
- Implemented media queries in a separate CSS file.
- Ensured the layout adapts smoothly to different screen sizes.

### **Task 4 – Interactive Button**
- Added a JavaScript button that dynamically changes background color.
- Introduced DOM manipulation and event handling.

### **Task 5 – API Integration**
- Used the Fetch API to retrieve and display JSON data from a public API (JSONPlaceholder).
- Rendered data dynamically inside the webpage.

### **Task 6 – Form Design & Validation**
- Created a styled registration form.
- Implemented client-side validation with JavaScript.
- Provided real-time user feedback for incorrect inputs.

### **Task 7 – Component-Based Styling (Bootstrap)**
- Integrated Bootstrap to enhance the UI using ready-made components.
- Refactored layout to ensure cleaner, structured, and responsive design.

### **Task 8 – SASS/SCSS Preprocessing**
- Created variables, nesting, and reusable styling with SCSS.
- Compiled SCSS into CSS and linked it to the HTML file.
